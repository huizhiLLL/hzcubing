# hzcubing

面向魔方玩家的社群成绩系统

成绩提交、榜单查看、选手主页、历史记录，以及官方 / 趣味 / 整活项目展示

前端 Vue 3，后端 Express + MongoDB。

## 主要功能

- 用户注册与登录
- 头像、昵称与个人资料维护
- 成绩提交、编辑与删除
- 分项目榜单与个人最佳
- 单项目 GR 打破历程查询
- 选手主页与历史成绩
- 管理员活动项目管理

## 项目结构

```text
src/        Vue 3 前端应用
server/     Express API 与 MongoDB 数据层
public/     静态资源
docs/       项目文档
```
