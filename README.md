# hzcubing

面向魔方玩家的社群成绩系统，提供成绩提交、榜单查看、选手主页、历史记录，以及官方 / 趣味 / 整活项目展示

前端使用 Vue 3 + Pinia + Vue Router + Vite，后端使用 Express + MongoDB。

## 主要功能

- 用户注册、登录与当前用户资料获取
- 头像、昵称、WCA ID、简介等个人资料维护
- 成绩提交、编辑、删除与个人历史记录
- 分项目单次 / 平均榜单，支持搜索、高亮当前用户和移动端卡片展示
- 个人主页、个人最佳、最近成绩和管理成绩入口
- 选手总览，支持搜索、排序和分页
- 首页社区统计与项目展示
- 单项目 GR 历程和近期破纪录信息
- 管理员整活项目管理
- QQ 绑定与 AstrBot 插件入口

## 项目结构

```text
src/        Vue 3 前端应用
server/     Express API 与 MongoDB 数据层
public/     静态资源
docs/       项目文档
```
